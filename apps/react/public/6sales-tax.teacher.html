User story

As a user, I want to calculate my cost (final price) of a pizza lunch by entering the retail price and selecting my province.

Steps

  1. As a developer, I want to log one province name (not abbreviation). This is a useful first step to understanding the data store schema (see rates.js)
  2. As a developer, I want to log every province name (not abbreviation)
    • As a developer, I want the DOM to display every province name (not abbreviation)
    • As a user, I want to select my province from a dropdown list.
  4. As a developer, when a province is selected I want to console.log the sales tax
  5. As a user, I want to input the pizza price to learn the cost based on my province
  6. As a user, I want to input the pizza price to learn the cost based on my province and display as currency.
