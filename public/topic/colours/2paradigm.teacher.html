Programming paradigm

Communication styles

Resources

  1. Wikipedia Programming paradigm

Technologies

  1. imperative: exess variable which is often easier to understand. For loops are common
    1. procedural: recipe with step by step approach
    2. object-oriented: Original React.js approach (class syntax)
  2. declarative: specific functions that may be called in any order. Interators are common
    1. functional: React Hooks recommended
    2. logic
    3. mathematical

Samples

	// imperative
	// output 1, 2, 3, 4, 5... 10
	for (const x = 1; x <= 10; i++) {
		console.log(x);
	}

	// declarative
	// output 1, 2, 3, 4, 5... 10
	Array.from(Array(10).keys()).map((index) => {
		console.log(index + 1);
	});

	// Or using variable names
	const plusOne = (index) => console.log(index + 1);
	const listOneToTen = Array.from(Array(10).keys());

	listOneToTen.map(plusOne)

	// splice, push, shift, unshift will manipulate a single Array
	// map, reduce, filter will read an array and output a new array
