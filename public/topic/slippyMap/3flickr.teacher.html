Slippy Map

Map with Flickr pins

Goal

As a user, I want to see pins of Vancouver beaches on a map. I want to see a photo in the pin details.

As a dev, Display a slippy map with a many pins representing photos found on Flickr. When user clicks on a pin, the photo will display in the info bubble.

Technologies

  • Flickr API (Service)
  • Mapbox GL JS API (Library)
  • JavaScript
  • HTTP Request with Fetch
  • CSS
  • HTML

Resources

  • Flickr API documentation for geo photos
  • Mapbox GL JS API
